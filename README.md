# EstateXpert

EstateXpert is an AI-based Real Estate Development & Investment Advisory System.

## Tech Stack
- Backend: Python (Flask – App Factory Pattern)
- Frontend: Flask-rendered HTML + Bootstrap
- Database: MySQL (XAMPP, manual phpMyAdmin execution)
- Analytics: R
- CMS Integration: WordPress
- AI/ML: Planned in later modules

## Project Discipline
- Modular, frozen structure after Module-0
- Flask Blueprints only (no new root apps)
- Manual database execution only
- Day-wise, micro-step execution
- GitHub is the single source of truth
